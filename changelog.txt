# Changelog & Error History

## v4.2.7
- **UI Bug Fix:** Memperbaiki konflik nama komponen (`Menu` dari Headless UI vs icon dari `lucide-react`) yang menyebabkan error rendering 'Passing props on Fragment' di `DashboardLayout`.

## v4.2.6
- **Headless UI Refactoring:** Mengganti custom AnimatePresence Overlay dan dropdown pada UI DashboardLayout dengan komponen `Dialog`, `Menu`, dan `Transition` dari `@headlessui/react`. Pendekatan ini mengatasi masalah a11y, menangkap fokus secara sempurna, dan memberikan perilaku responsif yang lebih native serta andal di Mobile maupun Desktop.
- **Automated Low Stock Alert:** Fungsi monitoring stok (`checkLowStockAndNotify`) berhasil diterapkan langsung pada `server.ts`. Sistem secara otomatis mengecek stok setelah transaksi berhasil diupdate, dan mengirimkan email peringatan ke Seller apabila terdapat sisa produk di bawah 5 item.

## v4.2.5
- **Admin Products Enhancement:** Penambahan tombol "Tambah Produk" untuk Admin dengan dukungan parameter form yang lengkap termasuk deskripsi (description).
- **Admin Products Low Stock Indicator:** Menambahkan highlight peringatan (visual indicator warning & border merah berkedip) di tabel & card view untuk stok produk di bawah 5.
- **Payment Methods Toggle:** Menambahkan panel khusus di `AdminSettings` untuk Superadmin guna mengatur visibilitas metode pembayaran (QRIS Dinamis, Statis, VA, Lainnya) secara real-time di halaman Kiosk Checkout. Metode Virtual Account (VA) secara default dinonaktifkan (disimpan).

## v4.2.4
- **Dashboard Stats Fix:** Menghapus label "trend" manual (12%, 8%) pada layar Admin Dashboard karena statik dan membingungkan.
- **Admin Revenue Breakdown Fix:** Memperbaiki algoritma fetch `transaction_items` sehingga mampu menampilkan rincian toko pengelola transaksi digital (PPOB) & Koperasi dengan benar di Breakdown Pendapatan Admin Dashboard.
- **Kiosk LCP Performance:** Menyesuaikan lazy loading gambar Catalog Kiosk, mencegah `eager` fetcher ditahan oleh attribute lazy (Mempercepat Lighthouse visual rendering di Kiosk).

## v4.2.3
- **Superadmin Fixes:** Memindahkan konfigurasi auto-upgrade Superadmin ('alfanza26@gmail.com') langsung ke level database (`supabase-schema.sql`) menggunakan blok DO $$ SQL yang dieksekusi secara native untuk menyelesaikan masalah bug trigger node. 
- **Admin Transaction Date:** Tanggal pada filter transaksi Admin di-layer agar tampil dalam format `DD-MMM-YYYY` (menggunakan helper `date-fns` `format`) sementara field aslinya tetap menggunakan Native Date picker.
- **PWA Manifest Improvements:** Memperbaiki file manifest PWA dengan menyertakan logo SVG, memastikan cache ukuran icon up-to-date, dan menyesuaikan warna `theme_color` agar sinkron dengan biru aplikasi.
