v4.8.0 - Autonomous Error Reporting & Category Fixes
--------------------------------------------------
### Fitur Baru & Optimasi:
- **[NEW] Autonomous Error Reporting:** Sistem kini secara otomatis menangkap error JavaScript, Promise rejections, dan log konsol. Dilengkapi dengan "Breadcrumbs" (jejak aksi user) dan metadata lingkungan teknis untuk mempermudah diagnosa oleh admin.
- **[FIX] Categories RLS & Schema:** Memperbaiki masalah "policy violation" saat menambah kategori dengan menambahkan kebijakan RLS yang tepat untuk Admin. Menambahkan kolom `slug` yang hilang pada tabel kategori.
- **[ENHANCEMENT] PWA UI Cleanup:** Menghilangkan tombol debug "Install App (Debug)" namun tetap mempertahankan banner instalasi PWA yang bersih dan profesional.
- **[ENHANCEMENT] Error Visibility:** Dashboard Admin Reports kini menampilkan detail teknis lengkap (Stack Trace & Breadcrumbs) untuk setiap laporan otomatis.

v4.7.1 - Excel File Import & Enhanced Bulk Tools
--------------------------------------------------
### Fitur Baru & Optimasi:
- **[NEW] Direct Excel Import (.xlsx):** Admin kini bisa mengunggah file spreadsheet secara langsung. Sistem akan membaca seluruh baris data dan memprosesnya secara otomatis menggunakan library `xlsx`.
- **[NEW] Hybrid Bulk Interface:** Peningkatan modal Bulk Import yang menawarkan dua opsi: "Upload File" (untuk data resmi) atau "Paste Data" (untuk pembaruan cepat).

v4.7.0 - Flash Sale Transformation & Scheduling
--------------------------------------------------
### Fitur Baru & Optimasi:
- **[NEW] Scheduled Flash Sale:** Menambahkan sistem penjadwalan (`start_time`). Tombol booking sekarang terkunci otomatis hingga waktu lelang dimulai.
- **[NEW] Real-time Countdown:** Portal buyer kini menampilkan hitung mundur (Hari/Jam/Menit/Detik) untuk aset yang akan datang.
- **[NEW] Admin CRUD Assets:** Admin kini memiliki kendali penuh untuk Menambah, Mengedit, dan Menghapus aset lelang secara langsung dari dashboard.
- **[NEW] Bulk Import Excel (Paste):** Fitur impor data masal menggunakan teknik Paste dari Excel. Otomatis memetakan kolom ItemName, AssetCode, SerialNo, Spek, dan Kelengkapan.
- **[NEW] Auto Brand Detection:** Sistem otomatis memberikan gambar generic berkualitas tinggi untuk produk Dell dan Lenovo jika foto manual tidak disediakan.
- **[ENHANCEMENT] Database Stabilization:** Pembaruan skema `sps_assets` untuk mendukung `start_time`, `final_price`, dan `winner_id` dengan kebijakan RLS yang lebih aman.

v4.6.1 - Admin Form Builder & Export Excel
--------------------------------------------------
### Fitur Baru & UI:
- **[NEW] Admin Form Responses:** Halaman khusus untuk melihat hasil respon formulir dinamis. Dilengkapi dengan tabel pencarian (Nama/NIK) dan modal detail untuk melihat jawaban lengkap anggota.
- **[NEW] Export to Excel:** Fitur ekspor data respon ke format `.xlsx` menggunakan library `xlsx`. Output dilengkapi dengan penyesuaian lebar kolom otomatis (auto-width), header yang rapi, dan penanggalan format lokal Indonesia.
- **[ENHANCEMENT] Google Forms-like UX:**
  - **Reordering Control:** Menambahkan ikon "Grip" dan tombol pindah (Up/Down) yang selalu terlihat untuk memudahkan penyusunan pertanyaan.
  - **Visual Feedback:** Peningkatan indikator fokus pada pertanyaan yang sedang diedit.
- **[FIX] Routing Admin:** Perbaikan 404 pada AdminFormBuilder yang disebabkan oleh ketidakkonsistenan case-sensitive pada path file.
