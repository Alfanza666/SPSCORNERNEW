# Changelog & Error History

## v4.8.0 — MINOR: Dynamic Form Portal & Advanced Form Builder
**Tanggal: 2026-05-17**

### Fitur Baru & UI:
- **[NEW] Portal Daftar Formulir:** Menambahkan halaman `/portal/forms` sebagai pusat akses formulir dinamis bagi anggota. Memungkinkan akses survei/pendaftaran mandiri tanpa harus melalui halaman program.
- **[NEW] Form Context Integration:** Portal kini menampilkan informasi jika sebuah formulir terkait dengan program tertentu (misal: Gathering/Family Day).
- **[ENHANCEMENT] Admin Form Builder (Google Forms-Style):**
  - **Inline Header Editing:** Edit judul dan deskripsi formulir langsung di canvas.
  - **Active Field Highlighting:** Penanda visual (blue border & left indicator) pada pertanyaan yang sedang diedit.
  - **Field Duplication:** Tombol sekali klik untuk menduplikasi pertanyaan yang sudah ada.
  - **Improved Option Management:** UI baru untuk menambah/menghapus opsi pada pilihan ganda (radio/checkbox/dropdown) secara individual (bukan lagi comma-separated).
  - **Shareable Links:** Fitur "Salin Link" di Dashboard Admin untuk mempermudah distribusi formulir mandiri.
- **[UI] Portal Navigation:** Menambahkan menu "Daftar Formulir" pada sidebar portal anggota untuk akses cepat.

### Perbaikan Sistem:
- **[FIX] Redirect Logic:** Memperbaiki alur navigasi pasca-submit formulir; pengguna akan dikembalikan ke daftar program jika mengisi pendaftaran program, atau kembali ke daftar formulir jika mengisi survei mandiri.
- **[FIX] Announcement Images:** Integrasi penuh Supabase Storage untuk upload poster pengumuman dari sisi Admin dan tampilan di Portal.
